# Scoring Rubric v2.0 — TSMC PI 盡職調查統一評分標準

> **版本差異**：v1（舊）將「獨家/深度綁定」列為 E ❌ 剔除類。v2.0 改為**漸進打分 + 合作狀況披露**。
> **適用範圍**：hv-research 所有 TSMC PI 盡職調查（2026-04 起）。

## 核心變動

| 面向 | v1 舊規則 | v2.0 新規則 |
|---|---|---|
| 合作衝突 | 🔴 Deep Bound → E ❌ 剔除不評分 | 維度 4 漸進打分（0-10），**仍進入大表** |
| 媒體熱度 | D 類獨立歸類（0/3, 1/3） | 取消，依實際 5 維度分數歸 B/C |
| 大表結構 | Top 15 / Backup 55 二分 | **單一大表 74 位**，分數 + 優先級標籤 |
| 合作揭露 | 備註欄散記 | **專欄「與外部公司合作狀況」** 統一披露 |

## 5 維度評分標準（0-10 分）

### 維度 1：研究軌跡可信度（權重 20%）

衡量學術深度、研究穩定性、國際視野。

| 分數 | 描述 |
|---|---|
| 9-10 | IEEE/ACM/AMS Fellow；頂會/頂刊一作穩定近 5 年；國科會傑出獎；h-index ≥ 40 |
| 7-8 | 頂會一作近 3 年 ≥ 3 篇；NSTC 一般計畫主持；h-index 20-40 |
| 5-6 | 中段期刊/會議；h-index 10-20；方向清晰但產出節奏不穩 |
| 3-4 | 產出零散；h-index < 10；或近 5 年方向漂移 |
| 0-2 | 無相關領域論文；或近 10 年零產出 |

### 維度 2：fab / semi 應用落地能力（權重 25%，最關鍵）

衡量研究成果與半導體生產實際需求的契合度。

| 分數 | 描述 |
|---|---|
| 9-10 | 方法已在 fab / 封測廠量產驗證；有技轉金 / 專利授權 / 商秘紀錄 |
| 7-8 | 原型級 PoC 完成；論文題目精準命中 fab 痛點（VM/APC/CD/yield）|
| 5-6 | 方法論可遷移但需 6-12 月 PoC；domain 邊界接近但非核心 |
| 3-4 | 方法論層次通用；半導體契合 < 30% |
| 0-2 | 無半導體論文；domain 完全不同（醫療/自駕/NLP/CG 等）|

**特別註記**：「跨 domain 遷移潛力」不等於「落地能力」。只有 WebFetch 找到半導體實證才能給 ≥ 5。

### 維度 3：Lab 深度與學生素養（權重 20%）

衡量可持續產出能力，以及 TSMC 人才管道 pipeline。

| 分數 | 描述 |
|---|---|
| 9-10 | Lab ≥ 20 人；頂會一作學生穩定；畢業生進 TSMC 有具名紀錄 |
| 7-8 | Lab 10-20 人；頂會學生產出；畢業生半導體業界流向清晰 |
| 5-6 | Lab 5-10 人；中段期刊學生產出 |
| 3-4 | Lab < 5 人；或學生產出以碩士論文為主 |
| 0-2 | 無可見 Lab；或已退休 / 借調 / 解散 |

**Haiku 快掃特別規則**：Lab 規模未找到 → 給 6（中性），並在理由欄標「Lab 規模待確認」。

### 維度 4：可接洽度 / 資源未被搶佔（權重 20%） **【v2.0 重寫】**

**舊版**：深度綁定 → 0 分 → 剔除
**新版**：漸進打分，**仍列大表**；所有已知綁定在「合作狀況」欄披露。

| 分數 | 情境 | 範例 |
|---|---|---|
| 9-10 | 完全自由 PI；無外部顧問/Chair/JDP | 江國寧、銀慶剛 |
| 7-8 | 無 TSMC 競業方綁定；有開放式國際合作 | 陳亮嘉（設備商中性）|
| 5-6 | 有非獨家顧問職 / Chair / 技轉廠商（非競業）| 陳智（Chemleader 技轉）|
| 3-4 | 有友廠（非核心競業）Chair / 半專屬 JDP | 鄭桂忠（TSMC-JDP）|
| 2 | 有競業方 Chair / 獨家顧問（MediaTek、聯發等）| 胡璧合（美光 Chair，非獨家）|
| 1 | 競業方獨董 / 長期獨家顧問 / Director | 張耀文（MediaTek 獨董）|
| 0 | 內部全職員工 / Corp Director / 借調中 | 張孟凡（TSMC Director）、彭文志（TSMC 借調處長）|

**關鍵原則**：低分**不代表剔除**。TSMC 若認為此 PI 技術獨特，仍可進入「觀察名單」，等狀況變動（獨董卸任、借調結束）立即啟用。

### 維度 5：長期合作價值（5 年回收潛力，權重 15%）

| 分數 | 描述 |
|---|---|
| 9-10 | 40-55 歲黃金期；研究軌道與 TSMC 5 年路線圖高度吻合；接班人梯隊清晰 |
| 7-8 | 45-60 歲；3 年可見共同成果；跨校合作網絡強 |
| 5-6 | 30-40 歲新銳或 55-65 歲資深；軌道匹配度中等 |
| 3-4 | < 30 歲 (未升等前) / > 65 歲（退休前）|
| 0-2 | 已退休；或軌道與 TSMC 5 年路線完全不吻合 |

## 加權總分計算

```
總分 = 維度1 × 0.20 + 維度2 × 0.25 + 維度3 × 0.20 + 維度4 × 0.20 + 維度5 × 0.15
```

## 優先級標籤

| 標籤 | 分數區間 | 意義 |
|---|---|---|
| **S** | 總分 ≥ 8.0 | 主打名單（第一波接洽目標）|
| **A** | 7.0 - 7.9 | 遞補首選 |
| **B** | 5.5 - 6.9 | 可觀察 / 單點合作 |
| **C** | < 5.5 | 方向偏 / 待激活 / 無急迫性 |

**特別標記**：維度 4 ≤ 2 → 在優先級後加 `🚩`（主管決策時需注意合作狀況欄）。

## 合作狀況欄位規範（新增）

**欄位標題**：「與外部公司合作狀況」

**記錄項目**：
1. **身份型綁定**：獨董 / Chair Professor / Director / Chief Scientist / 內部員工 / 借調
2. **顧問關係**：技術顧問 / 諮詢委員（含廠名、期間、獨家/非獨家）
3. **JDP / 產學合作**：合作廠商 + 合約期間
4. **技術移轉 / 授權**：技轉對象 + 金額（若公開）
5. **商業秘密綁定**：若前職為競業方且有明確商秘條款，標 🔴

**撰寫範例**：
```
張耀文: MediaTek 獨董 (2024/05-)；前 NTU EDA 中心主任；競業衝突高
鄭桂忠: TSMC-NTHU JDP 教授；ITRI 技術長 (2017-)；台達飛雁特聘 2022
胡璧合: 美光 Chair Prof. (2024-，非獨家)；TSMC PhD Scholarship × 3 屆；Lam Research 合作
江國寧: 無任何外部綁定；ASME/iMAPS 學會體系
```

**空白處理**：若真的無任何外部合作，填「無已知外部綁定」— 不得留空。

## 常見校準誤區（v2.0 繼承 v1.1）

1. **Agent 過度樂觀**：Haiku 掃完往往高估 0.5-1.5 分，我要手動校準。年齡/Lab 規模/頂會產出未核實前 → 扣 0.3-1.0
2. **不因主管點名而加分**：嚴格按 Rubric，即使主管 email 提到也不上推
3. **系祖追溯**：若發現新進 PI 是某大老的弟子（如鄭芳田→謝昱銘），獨立打分，避免因師承光環高估
4. **技職院校 quota**：NTUT/NTUST 不應因非 4 大就扣分；反而 fab 附近地緣與應用導向可能加分
5. **名氣 ≠ 契合度**：李宏毅 h-index 高但半導體 0 篇 → 維度 2 = 1，不應給面子分
