# 美瑛（Biei）景點與餐廳深度盤點

> 用途：日本北海道四城旅遊研究 — 美瑛分冊
> 對象：自由行台灣旅客，行程決策參考
> 編寫日期：2026-05

---

## A. 城市定位

| 項目 | 內容 |
|---|---|
| 行政區 | 北海道上川綜合振興局 上川郡美瑛町 |
| 地形 | 十勝岳連峰山麓的波狀丘陵地，標高 200–500 m，農田呈拼布狀 |
| 人口 | 約 9,000 人；面積 677 km²（札幌的 60%） |
| 最近大城 | 旭川（JR 普通車約 35 分鐘 / 自駕 30 分） |
| 從札幌 | 自駕約 2.5 小時（道央自動車道→旭川北 IC）；JR 特急 + 普通約 2.5 小時 |
| 觀光列車 | **富良野‧美瑛 Norokko 號**（夏季 6–9 月限定，旭川 / 美瑛 / 富良野 / 薰衣草花田站） |
| 別稱 | 「丘之町」、入選日本最美村莊聯合會（NPO「日本で最も美しい村」連合）創始成員 |

**為何出名**
1. **前田真三攝影**：1971 年起在美瑛丘陵長期創作，1987 年於舊千代田小學設立「拓真館」，使美瑛丘陵景觀進入大眾視野。
2. **昭和廣告聖地**：1972 年日產 Skyline「KEN ＆ MARY」、1976 年日本菸草「Seven Star」、1977 年日產 Cedric「親子の木」、1978 年「Mild Seven の丘」相繼成為廣告場景樹群。
3. **Apple iMac 開機畫面 / Mac OS X Snow Leopard 桌布**：「青い池」由白金溫泉湧出的鋁成份與美瑛川水混合產生膠質微粒，散射形成獨特乳藍色，2012 年因 Apple 內建桌布而爆紅。

**季節差異**

| 季節 | 月份 | 重點 |
|---|---|---|
| 春末 | 5月 | 殘雪 + 麥苗，淡季（花田尚未盛開） |
| 初夏 | 6月 | 馬鈴薯花、油菜花、青池葉色最翠綠 |
| 盛夏 | 7–8月 | 四季彩之丘 / 薰衣草 / 拼布之路全色 |
| 初秋 | 9月 | 麥穗金黃 + 收割 |
| 晚秋 | 10月底–11月初 | 白樺金黃；月底起花田收場、青池ライトアップ開幕 |
| 冬 | 12–3月 | 雪原、青池夜間燈飾、拓真館部分閉館 |

## D. 行程節奏建議

### D1. 自駕一日遊

| 出發地 | 可行性 | 動線建議 |
|---|---|---|
| 札幌 | ❌ 不建議單日 | 來回 ≥ 5 小時車程，丘陵停留只剩 3 小時，疲勞度高 |
| 旭川 | ✅ 最合理 | 09:00 旭川 → 09:45 拼布之路（北西の丘 / KEN&MARY / 七星）→ 11:30 美瑛駅午餐 → 13:00 超廣角之路（四季彩の丘 / 新栄の丘）→ 15:30 青い池 + 白鬚の滝 → 17:00 拓真館或回旭川 |
| 美瑛宿泊 | ✅ 最從容 | 兩個半日切分，傍晚拍夕陽（新栄の丘）/ 冬季拍青池ライトアップ |

### D3. 美瑛 + 富良野 2 日最簡路線

```
Day 1 (美瑛主軸)：旭川 → 美瑛駅 → 拼布之路 → 駅前午餐 →
                  超廣角之路 → 青池 + 白鬚の滝 → 美瑛宿
Day 2 (富良野順遊)：美瑛宿 → 拓真館 → 國道 237 南下 →
                  富田農場（薰衣草，7月）→ 唯我独尊午餐 →
                  富良野駅 / 旭川機場
```

### D4. 雪季造訪重點

- **必看**：青池ライトアップ（10/24–4/22）、雪原拓真館（11月開始 12月積厚最美）
- **避開**：1月下旬–4月中拓真館冬季休館；2月最冷可達 -25°C
- **駕駛門檻**：冬季雪地需 4WD + 雪胎，無經驗者**強烈建議改包車或美遊バス**

## E. 注意事項

| 項目 | 內容 |
|---|---|
| **「香煙廣告丘」最新狀態** | 哲學の木 2016/2 已伐除；Mild Seven 之丘防風林已間伐稀疏；**七星之木旁白樺並木 2025/1 已伐約 40 株**（樹本體保留）；KEN&MARY 之木與七星之木健在 |
| **春末 / 秋末為何淡季** | 5月：薰衣草、彩虹花田尚未開花，山頂仍殘雪；10月底：花田已收割，青池ライトアップ尚未開幕（4月底結束、10/24 起新季） |
| **自駕路況** | 鄉間道路無紅綠燈但起伏大，農機車隨時駛出；冬季雪地駕駛能力門檻高，無經驗者勿冒險；夏季注意「四季彩之丘」周邊單向道與停車秩序 |
| **農地拍照禮儀** | 樹群與花田絕大多數為**私有農地**，禁止跨越柵欄、踩踏作物、停放於耕作通道；違反導致哲學の木被伐、白樺並木消失，務必僅在道路 / 公共展望點拍攝 |
| **薰衣草 vs 美瑛花季時間差** | 富良野薰衣草盛開 **7月中–7月底**；美瑛四季彩之丘彩虹花田盛期 **7月–8月底**（種類多時序錯開），兩地差 1–2 週峰值 |
| **能力限制聲明** | 本研究基於公開網路資料（觀光協會、旅遊媒體、食評網），餐廳營業時間 / 公休日 / 預約規則可能隨季節變動，**出發前務必再次確認官方公告**；特別是冬季店家經常臨時休業 |
