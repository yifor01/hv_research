# 政大 (NCCU) 統計方法論補強：工業統計 × AI 教授盤點

**產出日期**：2026-04-22
**補強範圍**：統計學系、應用數學系（統計/最佳化方向）、科技管理與智慧財產研究所、心理學系（量化/統計方法）

## 六、政大統計系在本案的「方法論支援」角色定位

```
政大統計系
│
├── T1 支援：SPC/Change Point 方法論（鄭宇翔）
│   ↓ 應用：半導體 In-Line 製程監控、APC 系統
│
├── T3 支援：Causal Inference RCA（陳立榜）
│   ↓ 應用：良率根因分析、製程異常診斷
│
├── T3 支援：Bayesian Reliability/DoE（黃子銘）
│   ↓ 應用：OQC、設備壽命預測、YMS
│
└── T7a overlay：Bayesian DL UQ（翁久幸）
    ↓ 應用：不確定性量化 for Deep Learning 模型
```

**結論**：政大統計系雖無半導體 domain，但**純方法論 PI** 可作為「**方法論顧問軌**」——即在工研院/業界已有 domain 資料的前提下，由政大統計 PI 提供統計方法設計、模型驗證、論文共同掛名服務。建議以下分工：

| 角色 | 單位 |
|------|------|
| 半導體 domain + 資料 | 業界合作方（台積電、聯電等） |
| AI 工程 / DL 實作 | 政大資科（蔡銘峰、劉昭麟等） |
| **統計方法論設計** | **政大統計系（鄭宇翔、陳立榜、黃子銘）** |
| 制度對接 / 跨校聯合提案 | 政大 IAIC 中心（李蔡彥校長） |
