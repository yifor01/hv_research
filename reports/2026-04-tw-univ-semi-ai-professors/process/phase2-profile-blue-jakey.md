## 1. 技術契合度

### 1.1 現行研究題目（2023–2026）

根據個人頁 jakeyblue.github.io 與 NTU Scholars：

**主軸 A：AI-Empowered Process Control**
- Physics-Informed Run-to-Run（R2R）Control — 將設備即時健康狀態嵌入 APC 迴路
- Equipment Deterioration Modeling — 設備退化建模與故障根因診斷（Hidden Markov Model 框架，ISSM 2024 最新投稿）
- Virtual Metrology — 以 Gaussian Bayesian Network / Structure Data-driven 建構量測替代模型

**主軸 B：Singular Data Analytics**
- 不平衡資料分析（Imbalanced Data Analytics）
- 可解釋 AI（Explainable AI）for manufacturing
- 時間序列分析與需求預測（Demand Planning）

**副軸：跨域醫療 AI（2024–2026）**
- 腹腔鏡手術影像行動辨識（Laparoscopic Cholecystectomy Competency Assessment）
- 合作機構：台大醫院 Hsin-Chu Branch；已有 2 篇 *Surgical Endoscopy* 期刊論文（2025/2026）
- 此線與半導體無關，是獨立跨域研究，**非隱形綁定**，但佔用 PI 注意力

## 4. 合作優缺點 & 建議切入方式

### 4.1 優點

1. **副教授年輕活力**：2019 年入職，2022 升等，正值研究爆發期，有動機接新計畫
2. **唯一同時具備 TSMC 工程師 + 法國 APC 學術訓練 + NTU AI 方法論三重背景**的教授，方法論與廠端落地知識並重
3. **Mines Saint-Étienne 國際管道**：合作可借力法歐半導體製造學術網絡（NXP/ST 資料管道）
4. **T1（APC）× T3（Yield）× T6（Manufacturing Ops）三線同時命中**，比單一領域教授布局範圍更廣
5. **未被大廠獨佔**：無冠名講座或排他性中心主任頭銜，合作彈性大

### 4.2 缺點 / 風險

1. **論文產出節奏偏慢**：2022–2024 僅一篇 ISSM 會議 + 兩篇醫療期刊，在半導體核心領域產量不及簡禎富；H-index 13 相對中等
2. **開源 / 工具化能力待確認**：無公開 GitHub repo，工程落地能力需直接詢問
3. **醫療 AI 是否持續膨脹**：若 2025–2026 年再多 3 篇醫療論文，表示半導體主軸恐讓位
4. **Lab 規模中等**：學生人數不如簡禎富 / 陳安等大 Lab，大規模數據清洗與工程開發人力有限

### 4.3 與簡禎富的差異化（師承關係分析）

**簡禎富（Chen-Fu Chien）**：清華大學 IEEM；TSMC on-leave 副廠長級（2005–2008）；主持清華─台積電卓越製造中心；論文偏決策分析、排程、工廠 KPI；H-index 49。

**藍俊宏（Jakey Blue）**：NTU IIE；TSMC 一年基層工程師 → 法國 Mines 學術訓練；主攻設備健康監控與 APC；方法論偏 ML/統計物理建模；H-index 13。

**是否師生關係**：無直接證據。兩人分屬 NTU（Blue）與 NTHU（簡）不同學校，Blue 的 PhD 指導教授為 NTU 機械系 **Argon Chen（陳安）**（共同發表 2009–2021 論文多篇可佐證），與簡禎富無師承連結。二者為互補而非競爭關係。

### 4.4 可具體合作題目 3 個

**題目 A（T1 APC）：Physics-Informed APC 結合 LLM 異常解釋**
- 藍的 Physics-Informed R2R 框架 + 異常診斷能力，可疊加大語言模型生成自然語言「製程漂移報告」
- 切入方式：提供廠端真實 Run-to-Run 殘差資料，共同設計 ExplainableAI 模組

**題目 B（T3 Yield）：不平衡資料半導體 Yield 分類**
- 量產良率資料嚴重類別不平衡（良品 >> 不良品），Blue 的 Imbalanced Data Analytics 直接命中
- 切入方式：提供晶圓圖（wafer map）失效資料，合作開發 Few-Shot / SMOTE-Advanced 方法

**題目 C（T6 製造排程）：設備健康狀態嵌入動態排程**
- Blue 2018 年 CIE 論文（設備健康 + 排程整合）已有方法基礎；2024 ISSM 論文延伸到 HMM 設備狀態預測
- 切入方式：接 ISSM 2024 後續，合作提交 TSMC 台大聯合計畫（TSMC Academic Program）
