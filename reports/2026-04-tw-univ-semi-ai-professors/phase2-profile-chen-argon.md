## 3. 合作優缺點 & 建議切入方式

### 3.1 優點

| 優勢面向 | 說明 |
|---------|------|
| SPC/統計製程控制方法論奠基者 | EWMA/CUSUM 設計的 h-index 級代表作，在台灣半導體學界具歷史地位 |
| 產學轉化經驗豐富 | 從基礎研究→FDA 認證的完整軌跡（生醫），相同能力可反向用於半導體 AI 工具 |
| 國際計畫 PI 資歷 | SRC、ISMT、DARPA 計畫經驗，了解美國半導體研究生態 |
| 統計基礎紮實 | 近期 arXiv 論文仍在處理 multivariate classification、relative importance 等方法論問題，可銜接 ML 可解釋性 |
| 跨域視角 | 將半導體方法移植到醫療，反過來亦可將醫療 AI 技術反哺製造業 |

### 3.2 缺點 / 風險

| 風險面向 | 說明 |
|---------|------|
| 研究重心已轉生醫 | 近 3 年產出集中超音波影像、OSA 診斷，半導體 SPC 已非主攻方向 |
| Lab 規模偏小 | 約 3 名學生，投入產能相對有限 |
| DL 整合程度不明 | 近期論文以傳統統計為主，未見 LSTM/Transformer 等與半導體製程 AI 結合的論文 |
| 年資深、轉型中 | 博士畢業估計 1990 年代，現為資深教授；研究重心已在轉型，半導體方向可能是歷史積累而非未來投資 |
| 近期半導體合作紀錄薄弱 | 無直接近期業界合作公開紀錄，影響合作說服力 |

### 3.3 建議合作型態

**推薦：主題型顧問合作（型態 B）**

> 以方法論諮詢為核心，請陳正剛就 SPC×ML 融合方案提供統計嚴謹度評估，而非期待他帶頭推進半導體 AI 研究。

**理由：** 陳正剛的最大價值在於對 SPC/統計製程控制的深厚方法論功底，以及對台灣半導體業的歷史連結。但他的研究精力已在生醫領域，要求他轉回半導體主軸需要額外動機（如明確計畫資金）。適合以「統計審閱者/顧問」角色切入，而非「共同 PI」。

### 3.4 具體可切入題目

**題目 1：SPC + Deep Learning 的可解釋性橋接**
- 現有 deep learning 異常偵測（如 LSTM-based SPC）缺乏統計可解釋性
- 陳正剛在 relative importance measures 和 multivariate classifier 的近期工作，可直接銜接
- 合作形式：方法論共著，陳提供統計 framework，合作方提供製程資料

**題目 2：Equipment Health Index 的多模態融合**
- 陳 2009 年「Recipe-independent indicator for tool health diagnosis」是早期里程碑
- 延伸：整合 sensor data、process recipe、maintenance log 的多模態健康指數設計
- 與台積電或聯電的既有數據通道切入

**題目 3：小樣本場景的統計製程控制**
- 陳近年在醫療領域的「data efficiency / 小數據集」方法論，可直接遷移至半導體 APC
- 晶圓製程換型、新設備 burn-in 等場景天然缺乏大量訓練樣本

## 4. 風險評估

| 風險類型 | 評級 | 說明 |
|---------|------|------|
| 綁定風險 | 🟡 中等 | Lab 規模小（~3 人），研究產出集中；但因跨生醫/統計/製造，不至於單一依賴 |
| 執行風險 | 🟡 中等 | 研究重心已在轉型，半導體方向需要明確計畫誘因；退休時程未知但年資深 |
| 方法論新穎性 | 🟡 中等 | 歷史 SPC 方法論典範地位高，但 2022–2025 無直接 SPC×AI 半導體論文；新穎性存疑 |
| 產業連結 | 🟡 中等 | 歷史合作強（多家半導體公司），近期無公開紀錄；需直接確認現有合作狀況 |

**綜合評估：Tier-S #3 定位合理，但需確認研究動機是否仍在半導體方向。** 相較 Phase 1 評估，深度 profile 後發現陳正剛的半導體研究實質活躍程度低於預期，生醫已成其近年主線。建議在正式合作接洽前，先確認他是否有意回歸半導體 AI 方向（例如透過 NSTC 新計畫申請或與台積電的新合作動機）。
