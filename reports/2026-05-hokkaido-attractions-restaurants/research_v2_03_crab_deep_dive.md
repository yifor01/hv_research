## v2.6 北海道螃蟹深章（5月中聚焦）

v1 對螃蟹的處理只有札幌 1 家「かに本家」附帶提及，**這對「五月來北海道想吃蟹」的旅客遠遠不夠**。本章從蟹種旬曆 → 4城螃蟹店家分布 → 蟹宅配回國攻略 → 防雷常識，做一次性總整理。

### v2.6.1 北海道四大蟹種旬曆（5月中關鍵判斷）

北海道商業流通最常見的螃蟹有 4 種，名稱常被旅遊網站混用，先釐清基礎：

| 蟹種（中 / 日 / 英） | 體型 / 特徵 | **5月中是否當令** | 主要產地（道內） | 漁期 |
|---|---|---|---|---|
| **帝王蟹** タラバガニ Red king crab | 巨大（甲殼 25 cm+）、腳長味淡 | ❌ **非當季**（流通主峰 11-2月，5月多冷凍貨或進口） | 主為俄羅斯產進口；道內鄂霍次克少量 | 主漁期 11-2月（偶有春期） |
| **松葉蟹 / 楚蟹** ズワイガニ Snow crab | 中型、腳細甜味濃 | ❌ **禁漁期**（漁期 11/6-3/20，雌限 11-1月） | 鄂霍次克、留萌、後志 | **5月中市面流通幾乎全為冷凍** |
| **毛蟹** 毛ガニ / ケガニ Horsehair crab | 小型（甲殼 8-12 cm）、蟹膏濃郁 | ✅ **當令**（春期鄂霍次克 / 宗谷產進旺季） | 鄂霍次克（雄武 / 網走 / 紋別 / 斜里）+ 宗谷（稚內 / 枝幸） | **春 3-6 月（鄂霍次克 + 宗谷）→ 夏 7-8 月（噴火灣）→ 秋 9-10 月（釧路 / 根室）→ 冬 12-2 月（十勝）** 全年輪播 |
| **花咲蟹** 花咲ガニ Hanasaki crab | 中型多刺、味濃近龍蝦 | ❌ **非當季**（旬為 7-9月）| 根室半島、納沙布岬周邊 | 7-9 月為主 |

**單句結論（5月中）**：北海道 5 月中**唯一鮮活當令的本地蟹是「毛蟹」**，且應**優先選鄂霍次克產**（春期主產地）；タラバ / ズワイ / 花咲 在 5月中不是流通最佳狀態。

### v2.6.2 為什麼旅遊網站常誤導

許多繁中旅遊文章寫「來北海道一定要吃三大蟹（タラバ / ズワイ / 毛）」——這個說法成立但有兩個前提常被略過：

1. **「三大蟹」是指北海道整年市場中最有名的三種，不是任何時點都當令的三種。** 5月中鎖定毛蟹才能吃到道內鮮活；タラバ / ズワイ 的 5 月版本多為「冷凍解凍貨」或「俄羅斯進口」。
2. **餐廳與市場的差異**：高級螃蟹餐廳（かに本家 / かに將軍）有冷凍庫存與進口管道，**任何月份都能上「三大蟹套餐」**，但「當令鮮活」與「庫存鮮度」是兩個層次。價位差異主要也反映此差異（5 月 タラバ 比 12 月便宜 30-50%，因鮮度等級不同）。

**自由行決策建議**：5月中如果只想吃 1-2 餐螃蟹，**完全鎖定毛蟹**會比點「三大蟹拼盤」實際品質更高、CP 更好。想吃帝王蟹體驗，可在 5月中接受冷凍進口版（多數市場與餐廳的 5月帝王蟹仍是凍蟹切熟件型態）。

### v2.6.4 蟹宅配回國（台灣）攻略

要把活蟹 / 鮮蟹帶回台灣，**只能透過冷凍宅配 + 報關**，不能手提帶上飛機（檢疫禁止活體 + 鮮品）。實務上分兩類：

**A. 簡易方案：日本店家代寄至台灣**
- **適用店家**：札幌場外市場各大水產店（佐藤水產 / 函館長谷川）、新千歲機場「北海道どさんこプラザ」部分店家
- **流程**：店家代辦冷凍包裝 + 國際快遞（多採 EMS 或日本郵便）+ 收件人在台灣付海關稅 + 收件
- **時程**：發貨後 5-7 個工作日抵台
- **費用**：海鮮本身 ¥10,000-30,000 + 國際運費 ¥6,000-12,000 + 台灣海關稅（活水產品 5%-15%、依貨值）
- **限制**：日本國內配送 OK，但**直接國際配送台灣的水產品店家有限**；常需轉介專門海外配送服務

**B. 進階方案：透過台灣專業代購 / 代運**
- **適用對象**：要買多隻 / 高單價（如帝王蟹整隻 ¥20,000+）的旅客
- **流程**：在日本超市 / 市場購買 → 寄到台灣代運倉庫（東京 / 大阪 / 北海道皆可）→ 代運併單冷凍空運 → 台灣海關 → 宅配收件
- **服務商**：JapanTo / 通泰 / 樂天通配 / 日本 Tenso 部分支援冷凍
- **時程**：發貨後 7-14 工作日（依代運倉庫收件節奏）
- **限制**：每趟以「家庭自用合理數量」為原則，超過 6 公斤海關可能扣關退運

**C. 最簡方案：放棄帶回，當地吃飽**
- 認真執行：5月中在札幌住 2 晚，安排 1 晚かに本家 / 氷雪の門 螃蟹懷石、1 餐市場毛蟹定食、1 餐居酒屋蟹味噌湯——已經是「整趟蟹體驗」上限。**寄回國的成本通常比再來一趟北海道還高**。

### v2.6.5 防雷常識（5月中特別注意）

1. **冷凍 ≠ 不新鮮**：日本冷凍技術（-60°C 以下急速冷凍）保鮮度極高；タラバ / ズワイ 在 5月吃冷凍版本不是「踩雷」，但**價格不該比 12月便宜太多就是冷凍貨注意**。
2. **「活蟹」與「活造」差別**：活蟹（活け蟹）= 點餐前活體狀態；活造（生造） = 點餐後生食或半生食處理（多為毛蟹刺身、ズワイ 蟹腳生造）。**5月中 ズワイ 活造幾乎不可能**（禁漁期），看到聲稱「活ズワイ刺身」的店要警覺，可能是冷凍解凍貨。
3. **缺腳蟹 / 訳あり蟹**：完整蟹腳 8 隻、缺 1-2 隻就被列「缺腳蟹」半價賣，**味道與營養完全相同**——5月中市場想低成本吃蟹，缺腳毛蟹 ¥2,500-4,000 是好選擇。
4. **「市場食堂海鮮丼加蟹」CP 比餐廳高**：滝波食堂 / 海鮮処 魚屋の台所等市場食堂，加單款毛蟹（蟹身或蟹腳片）¥800-1,500 即可——比進蟹專門店點蟹腳便宜一半。
5. **退稅 + 宅配同店辦**：場外市場大型店家可一次處理「退稅 5-10% + 國際宅配」；單筆買到 ¥10,000 以上才符合退稅門檻。
6. **拼裝禮盒注意「真空 / 急冷 / 解凍要求」**：作為伴手禮買的「蟹腳禮盒」多為已蒸熟 + 急速冷凍 + 真空包裝，**回國後從冷凍庫拿出當天必吃完**，反覆解凍會嚴重影響口感。
