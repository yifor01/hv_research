## 5. 橫向分析 3：行程設計矩陣

行程設計的真正難題不在「有哪些景點」，而在「有限時間下如何取捨」。本節用三個維度切組合：天數、季節、自駕與否。

### 5.1 不同天數的城市組合建議

| 天數 | 推薦組合 | 不建議 | 理由 |
|---|---|---|---|
| **3 天 2 夜** | 札幌 + 小樽 | 加函館或美瑛 | 移動時間吃掉 1 天，剩下 2 天無法深度任何城市 |
| **4 天 3 夜** | 札幌 + 小樽 + 美瑛（旭川中繼） | 札幌 + 函館（移動單向就 0.5 天） | 美瑛可一日來回（住旭川）；函館需 2 天才划算 |
| **5 天 4 夜** | 札幌 + 小樽 + 函館（東西雙雄） | 4 城全套 | 最常見組合：D1 札幌、D2 小樽日歸、D3 移動函館、D4 函館全天、D5 飛回 |
| **6 天 5 夜** | 札幌 + 小樽 + 函館 + 美瑛 | — | 4 城全套的入門天數，但每城只能淺嚐 |
| **7 天 6 夜** | 4 城 + 富良野延伸 | — | 黃金組合，每城 1.5 天 + 1 天移動 buffer |
| **10 天 +** | 加入登別 / 知床 / 道東 | 札幌停留超過 3 晚 | 札幌 3 晚已是極限，再多就應外擴 |

**關鍵判斷點**：

- **「能不能加美瑛」的紅線是 4 天**：3 天連美瑛都別想（自駕門檻 + 旭川中繼 + 季節風險）；4 天可以但需自駕。
- **「能不能加函館」的紅線是 5 天**：4 天加函館等於 50% 時間在移動；5 天才剛好。
- **3 天的最佳組合永遠是「札幌 + 小樽」**：JR 30 分接駁、不需自駕、餐飲密度最高、最 user-friendly。

**單句決策**：

- **目標雙雪祭**：2 月 7–14 日札幌 + 小樽（窗口期不到 1 週）
- **目標櫻花**：4 月底函館為主場（其他三城櫻花期落差太大）
- **目標花田**：7 月下美瑛 + 富良野（薰衣草 + 彩虹花田同時）
- **目標楓葉**：10 月中下 4 城都好（次旺季氣溫舒適）
- **目標海膽**：6–8 月函館（道南馬糞海膽極盛、新鮮活海膽）

### 5.3 自駕 vs 鐵路的選擇

| 城市 | 鐵路可行性 | 自駕必要性 | 結論 |
|---|---|---|---|
| 札幌 | 地下鐵 + 路面電車覆蓋 90% 景點 | 0 | 完全不必租車 |
| 小樽 | JR + 步行 100% 覆蓋 | 0 | 完全不必租車 |
| 函館 | 市電 + 公車 + 步行 95% 覆蓋；大沼國定公園需 JR | 低 | 不必租車（除非加大沼） |
| 美瑛 | JR 只到美瑛站 + 美遊バス / 包車 / Norokko 號 | **高** | **強烈建議自駕**，否則拼布之路 / 超廣角之路兩半天行程拉不開 |

**從新千歲機場開始的兩種路線哲學**：

- **純鐵路派**：CTS → 札幌（JR 快速 Airport 37 分）→ 札幌市區地下鐵 → 小樽 JR → 函館 JR 北斗 → 函館市電 → 函館飛 CTS。**全程 0 自駕**，但放棄美瑛。預算友善（JR Pass 7 天 ¥27,430）、無雪地駕駛壓力。
- **混合派**：札幌期間用地下鐵 + JR 跑小樽；旭川 / 美瑛期間租 4WD 自駕；函館期間還車 + 飛機。**租車成本 ¥6,000–10,000/天 + 高速公路費**，但能完整跑美瑛。雪季駕駛能力有門檻（建議至少有 1 次以上雪地駕駛經驗）。

**單句決策**：第一次來 / 不會雪地駕駛 → 純鐵路派 + 放棄美瑛；攝影 / 自駕老手 / 想完整跑 → 混合派 + 接受 +¥30,000–50,000 / 趟租車成本。
