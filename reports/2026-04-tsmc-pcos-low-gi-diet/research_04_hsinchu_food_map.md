# 竹科 TSMC 工程師外食場景 PCOS / 低 GI 友善地圖

**對象**：25–30 歲 TSMC 女工程師（新竹科學園區 / 竹南），體重標準、預算 NTD 500/餐
**主訴**：PCOS（多囊性卵巢症候群）→ 痤瘡 + 落髮 + 體重難降 + 經期不規律
**研究日期**：2026-04
**研究方法**：Dcard / PTT / Google Maps / 部落格 / 營養師專欄 多源彙整 + 「能力限制標註」原則

## 0. 核心策略 TL;DR

PCOS 飲食三大鐵則（先記住，後面所有點法都圍繞這三條）：

1. **低 GI / 控制精緻澱粉**：白飯、白麵、麵包、含糖飲料、勾芡 → 全部視為「高傷害」；改吃糙米、紫米、五穀、藜麥、地瓜
2. **高蛋白佔每餐 1/3**：體重 (kg) × 1.5–2 g/day（55 kg 女性 → 每天 85–110 g）；雞胸、雞腿、魚、蛋、豆腐、無糖豆漿
3. **進食順序**：菜 → 蛋白質 → 主食（最後才碰澱粉，可降 GL 30%+）

避雷四象限：
- **糖**：手搖飲、果汁、優格沙拉醬、糖醋、照燒、麵包、御飯糰中的「肉鬆/起司」口味
- **油炸**：炸雞、薯條、天婦羅、炸豬排、雞排
- **加工肉**：火腿、培根、香腸、熱狗、火鍋餃類
- **勾芡**：燴飯、羹湯、糖醋、酸辣湯（澱粉 + 糖二合一）

附帶補充劑（飲食外）：肌醇 myo-inositol（早晚各 2 g）、維他命 D3、鋅、Omega-3 — 但這份報告只聚焦在「外食點法」，補充劑請見研究 #02。

### 1.4 員工餐廳「應急三招」

**情境 A：今天只有油膩菜色**
→ 拿白飯半碗 + 雙份燙青菜 + 茶葉蛋 2 顆 + 味噌湯，蛋白質從茶葉蛋補

→ 員工餐廳的「水果區」拿芭樂 / 蘋果 / 奇異果 1 顆，配無糖優格（自帶或便利商店買），可滿足甜味需求又不爆血糖

## 2. 7-11 / 全家 PCOS 友善早餐清單（最重要章節）

### 2.1 為什麼要吃早餐？（給「不想吃早餐」的工程師）

PCOS 患者**最容易在早上發生胰島素抗性高峰** — 空腹太久 → 中午暴吃 → 血糖飆升 → 雄激素飆升 → 痘痘 + 落髮加劇。即使不餓，也要在 9:00 前吃進「至少 15g 蛋白質 + 1 份膳食纖維」。

**最低門檻**（90 秒搞定，不用準備）：
> 1 顆茶葉蛋（7 g 蛋白質）+ 1 杯無糖豆漿 450ml（13–17 g 蛋白質）= 65 元，蛋白質 ≥ 20 g，0 糖

### 3.2 點餐 SOP（不論哪一家都通用）

1. **主食換成糙米 / 五穀 / 藜麥 / 紫米** — 換不了就「半份米飯 + 多一份青菜」
2. **主菜避開糖醋 / 照燒 / 蜜汁系**，選清蒸 / 香煎 / 烤
3. **指定「醬料另外」** — 糖份大多藏在醬汁裡
4. **主菜 1 份 + 蔬菜雙份** — 大多店家可加 30–50 元加菜

## 4. 五大晚餐情境的 PCOS 友善點法 SOP

### 4.1 健康餐：見上一章節，最不需要動腦

### 4.2 拉麵店降傷害點法（不戒掉但 ≤ 月 2 次）

#### 為什麼拉麵是 PCOS 地雷？
- 麵體：精製麵粉 → 高 GI（GI 60+）
- 湯頭：豚骨白湯熬製含大量飽和脂肪 + 鈉
- 配料：叉燒（紅肉 + 加工）+ 溏心蛋（加分）+ 海苔（OK）+ 玉米（高 GI）

#### 新竹常見拉麵店（連鎖 + 在地）

| 店名 | 地點 | 湯頭 | PCOS 友善建議 |
|------|------|------|---------------|
| **一風堂** | 巨城 | 博多豚骨（赤丸 / 白丸）| 點「赤丸」較刺激味蕾，麵量減半 |
| **屯京拉麵** | 巨城 | 東京豚骨醬油 | 醬油湯較豚骨清，可指定不加大蒜油 |
| **滝禾製麵所** | 新竹市區 | 蒜香 / 咖哩 / 麻辣 / 鹽味雞湯 | **鹽味雞湯**是首選，比豚骨低脂 |
| **三田製麵所** | 新竹巨城 | 沾麵為主 | 沾麵可控制吃多少湯（吸附較少）|
| **Hiro's らぁ麵 Kitchen** | 新竹 | 濃郁系 | 油脂高，建議單純喝湯 1/3 |
| **樂麵屋** | 新竹 | 客製化湯頭 | 可選清淡湯頭 + 不要油 |

#### 拉麵「降傷害五招」（依糖尿病營養師建議）
1. **湯頭選「醬油 / 味噌 / 鹽味」**（清湯 GI < 豚骨）
2. **麵量「少」或「半份」** — 大部分日系拉麵店都接受
3. **加點：溏心蛋 ×2、燙青菜、海帶、豆腐** — 蛋白質 + 纖維前菜
4. **湯不要喝完** — 鈉與脂肪集中在湯裡
5. **配茶（熱綠茶 / 麥茶）不配可樂 / 啤酒**

#### 點法範例（一風堂）
- 赤丸（無細麵 = 預設粗麵 = GI 略低）+ 半份麵 + 加溏心蛋 + 加海帶 + 不喝湯 + 熱綠茶
- 預估熱量 700 kcal（vs 標準版 1100 kcal）

#### 頻次建議
- **每月 ≤ 2 次**；月經前一週與排卵期當週**避免**（雄激素敏感期）

### 4.3 鐵板燒 PCOS 友善點法（比拉麵友善太多 ≤ 週 1 次）

#### 為什麼鐵板燒相對友善？
- 蛋白質份量大（菲力 / 雞胸 / 海鮮）
- 蔬菜可雙份
- 主食白飯可指定半碗或不要
- 油量比拉麵 / 油炸料理低

### 4.4 日式料理 PCOS 友善點法

#### 推薦排序（高 → 低）

**Tier 1：定食類（最佳）**
- **大戶屋**：海鮮定食（鯖魚、𩸽魚 280–430 元）+ 五穀飯 60 元 + 白飯減半
  - 點法：什蔬燴黑醋醬定食 / 炭烤鯖魚定食 + **主動換五穀飯** + 不要點甜點
- **銀座杏子日式豬排**：大里肌 / 腰內肉定食（炸物，但份量可控；偶爾吃 OK）
- **大魯閣 / 鳥どり**：燒鳥定食類（雞腿、雞胸串），蛋白質充足

**Tier 2：壽司類**
- **藏壽司 / 壽司郎 / 爭鮮**：
  - **可吃**：鮭魚 / 鮪魚握壽司、玉子壽司、鮭魚卵軍艦、生菜沙拉、味噌湯、茶碗蒸
  - **避開**：天婦羅壽司、肉鬆軍艦、玉米軍艦、起司軍艦、牛肉壽司（油脂高）、甜點 / 紅豆湯
  - 控制：**握壽司 ≤ 8 貫**（每貫米飯 ~10g，8 貫 = 約半碗白飯量）

**Tier 3：豬排 / 咖哩飯**
- **CoCo 壹番屋**：選「蔬菜咖哩」+ 飯量「少」（200g）+ 辣度 3+；避開炸物配料
- **銀座杏子豬排**：偶爾為之（≤ 月 1 次），點瘦肉腰內肉而非大里肌

#### 萬用搭配
- **茶碗蒸**（蛋白質 + 低熱量）→ 永遠加點
- **味噌湯**（發酵食品 + 海帶豆腐）→ 包含在套餐中
- **生菜沙拉**（要求「醬另外」，自己控制醬量）→ 加點

### 4.5 泡麵急救方案（萬不得已時）

#### 為什麼泡麵是 PCOS 最差選項？
1. 油炸麵體 → 高飽和脂肪 + 高 GI（GI 70+）
2. 調味粉包 → 鈉 1000–3120 mg（單包就佔每日 50–130%）
3. 無蛋白質、無纖維 → 純精製澱粉 + 油 + 鹽
4. 棕櫚油成分 → 加劇雄激素

#### 不得已要吃時的「升級方案」

```
原本：泡麵一包 → 預估熱量 500 kcal、蛋白質 8g、鈉 1500mg
升級後：
  + 雞蛋 ×2 顆（蛋白質 +14g）
  + 冷凍青菜 / 高麗菜絲一把（纖維 +3g）
  + 板豆腐半盒（蛋白質 +8g）
  + 油包只用 1/3、調味粉用 1/2
  + 倒掉一半湯後再吃
  + 配無糖豆漿 1 杯（蛋白質 +13g）
最終：熱量 ~600 kcal、蛋白質 ~43g、鈉減半
```

#### 品牌相對排序（要吃就選這幾款）
| 品牌 | 評級 | 原因 |
|------|------|------|
| 韓國辛拉麵（湯麵）| ★★★ | 麵體較有嚼勁、含少量蔬菜乾 |
| 統一肉燥麵（小包）| ★★ | 份量小、熱量上限低 |
| 滿漢大餐 | ★★ | 含真空料理包、蛋白質略多 |
| 統一麵 / 維力炸醬 | ★ | 純精製，避開 |
| 杯麵類 | ✗ | 鈉與油脂濃度最高 |

#### 頻次建議
- **每月 ≤ 1 次**；最好直接戒掉，改吃便利商店冷藏便當

### 5.4 應酬 / 聚餐場景

#### 火鍋（最 PCOS 友善的應酬選項）
- **湯底**：昆布 / 蔬菜 / 大骨 / 菌菇；避開麻辣 / 咖哩 / 沙茶 / 豚骨
- **下鍋順序**：青菜 → 主食 → 火鍋料 → 肉類（保留血糖緩衝）
- **食材**：海鮮 / 雞腿 / 牛梅花；避開貢丸 / 蛋餃 / 餛飩 / 火鍋餃類
- **主食**：冬粉 ✓ / 烏龍麵（少量）；避開王子麵
- **沾醬**：蔥 + 蒜 + 蘿蔔泥 + 醬油 + 辣椒；避開沙茶醬 / 芝麻醬 / 腐乳醬
- **飲料**：白開水 / 無糖綠茶；避開可樂 / 啤酒 / 果汁

#### 燒烤
- 點蛋白質：海鮮類（蝦、蛤、烤魚）、雞肉串、菲力牛排
- 避開：香腸 / 培根捲 / 五花肉 / 雞皮 / 起司類
- 蔬菜：杏鮑菇、青椒、玉米筍（玉米避開）
- 醬料：原味鹽烤 / 檸檬汁；避開照燒 / 甜辣醬
- 主食：避開白飯 / 麵類，多吃蛋白質就飽

#### 熱炒
- 點：炒青菜 ×2、清蒸魚、白切雞、蛤蜊湯、皮蛋豆腐
- 避開：糖醋系列、勾芡羹湯、魷魚螺肉蒜（高糖）、三杯雞（醬料糖份高）
- **酒精**：盡量推掉；若無法推掉，威士忌 / 紅酒 1 杯 < 啤酒 / 調酒（含糖）

#### 同事聚餐前一條應對腳本
> 「我最近在調身體，醫生叫我少吃精緻澱粉跟糖，所以飯不用幫我點，我多吃菜跟肉就好。」
> （PCOS 不需明說，「調身體」是台灣職場最廣為接受的說法）
