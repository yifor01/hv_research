# 健康狀況

> 最後更新：2026-04-27

## 核心健康框架

### Lean PCOS（多囊性卵巢症候群 - 瘦型）

- **診斷**：已確診 lean phenotype（BMI 正常 + 胰島素阻抗 + 雄性素過高）
- **核心病理**：下視丘-腦下垂體軸異常 + 神經內分泌驅動 + 慢性低度發炎，**不是**脂肪組織發炎為主的代謝負擔
- **族群盛行率參考**：亞洲女性 lean PCOS 約佔所有 PCOS 的 40%（高於西方）
- **常見共病風險**：
  - 橋本氏甲狀腺炎 25-30%（任何海洋來源 / 碘負荷需評估）
  - 痘痘 / 皮膚問題（已發生，見下）
  - 慢性疲勞 / 睡眠品質下降
  - 月經週期不規律
  - 備孕風險（生育年齡高峰，需考慮肌醇 + 葉酸時序）

### 痘痘 / 皮膚狀況

- **類型**：外油內乾 + 粉刺 + 內包痘
- **病理路徑（個人推估）**：IGF-1 → 5α-reductase → DHT → 皮脂腺 lipogenesis（PCOS 雄性素 + 胰島素阻抗共驅）
- **皮膚屏障**：有受損跡象（外油內乾為典型表現）
- **痘疤狀況**：未明確區分（需專業皮膚科鑑別 PIH vs atrophic scar）

### Lean Phenotype 對美容的特殊風險（v2 報告新發現）

- 臉部脂肪墊（buccal fat / malar fat / 太陽穴）天生較少於 normal-weight 女性
- 過度減重會加速臉部中段（蘋果肌、鼻唇溝旁）流失
- **BMI 必須維持 19-21，不應追求 < 18.5**
- 慢性壓力（cortisol）會推動脂肪重分布（皮下 → 內臟），加重臉部凹陷感

## 飲食框架

- **執行中**：低 GI 飲食（基於 PCOS 胰島素阻抗管理）
- **熱量配置**：不應走 caloric restriction（lean phenotype 不需要熱量赤字，反而傷害代謝與澎潤）
- **蛋白質目標**：1.2-1.6 g/kg/日（PCOS 建議區間，亦支持膠原合成 + 維持脂肪墊）
- **避開**：高 GI 食物、果糖含量高的飲品、奶蛋白（IGF-1 推升）、leucine-rich whey protein

## 補充品立場（截至 2026-04-27）

### Tier 1 / P1（PCOS 核心）
- Myo-inositol 4 g/日（最好搭 D-chiro 40:1）
- Vitamin D 3000 IU/日（目標血中 > 30 ng/mL）
- Omega-3 EPA+DHA 2 g/日
- 鋅 25-50 mg/日

### Tier 2 / P2（次要）
- NAC 600 mg × 2-3/日
- 鎂 200 mg
- Berberine（個案需要）

### 美容類（保澎潤 P2 加碼）
- 口服玻尿酸 HA 120-200 mg/日（首選 > 口服膠原）
- 口服膠原 5-10 g/日（純配方，Vital Proteins / Sports Research Unflavored）
- 避開含糖 / 含奶 / 含 ornithine / arginine / glutamine 複方膠原飲

### 反指徵（不應補）
- 大豆異黃酮高劑量
- DHEA / pregnenolone
- 高 leucine 乳清蛋白
- 高劑量 Vitamin A palmitate（孕期禁忌）
